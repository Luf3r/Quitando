# CODEX.md — Quitando

Use [`AGENTS.md`](./AGENTS.md) como instrução operacional autoritativa e [`PROJECT.md`](./PROJECT.md) como contexto inicial do projeto.

Fluxo esperado para qualquer tarefa:

1. inspecione o repositório, as specs e a fase atual do roadmap;
2. leia os contratos relevantes no documento de domínio;
3. escreva ou ajuste uma spec que falhe pelo motivo correto;
4. implemente a menor mudança válida;
5. execute testes focados e verificações aplicáveis;
6. revise domínio, constraints, autorização, concorrência e acessibilidade;
7. atualize documentação se uma decisão mudar;
8. relate mudanças, comandos executados, resultados e riscos.

Não trate este arquivo como uma segunda especificação. Novas regras pertencem a `AGENTS.md` ou aos documentos normativos.
